# L&L Painting Website

This is a static multi-page website and marketing starter system for L&L Painting.

## Files

- `index.html` home page
- `services.html` services page
- `gallery.html` gallery page
- `reviews.html` reviews page
- `about.html` about page
- `estimate.html` lead capture page
- `styles.css` shared site styling
- `script.js` mobile navigation behavior
- `marketing-system.md` ad copy, follow-up scripts, and weekly marketing plan

## Before You Launch

1. Replace the remaining domain and form email placeholders.
2. Swap the stock image backgrounds with real project photos.
3. Replace the sample reviews with real customer testimonials.
4. Update the form action in `index.html` and `estimate.html` to your real lead destination.
5. Connect the domain and publish to any static host.

## Quick Launch Options

- Netlify
- Vercel
- GitHub Pages

## Lead Form Note

The forms currently use `formsubmit.co` as a simple placeholder. If you want leads sent instantly to email, text, or a CRM, replace the `action` URL with your preferred service.
